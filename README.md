## Installation

```sh
yarn add react-native-clocks
yarn add @shopify/react-native-skia
```

## Usage

Analog:

```js
import { Clock } from 'react-native-clocks';

// ...

export default function App() {
  return <Clock />;
}
```

Digital:

```js
import { DigitalClock } from 'react-native-clocks';

// ...

export default function App() {
  return <DigitalClock />;
}
```
